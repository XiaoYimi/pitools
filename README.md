# 安装 (Installation)
```
  npm install -g pitools
```

# 选项 (Options)
```
  -V, --version                        output the version number
  -h, --help                           output usage information
```

# 命令 (Commands)
```
  init <template-name> <project-name>  基于选择的模板快速初始化项目
  list                                 查看当前可初始化的模板
```

# 注意

```
在初始化模板时, <template-name> <project-name> 这两个选项为必填项; 如果不填则会导致出错.
```

# notice
```
When initializing a template, < template name > < project name > are required; otherwise, an error will occur
```
